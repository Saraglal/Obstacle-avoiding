# Obstacle-avoiding
Obstacle Avoiding Robot is an intelligent device that can automatically sense the obstacle in front of it and avoid them by turning itself in another direction.

## Components
- ultrasonic sensor
- two gear motors
- H-bridge motor driver (L293D)
- 9v battery
- arduino uno
- breadboard
- wires
